# Changelog — 2026-06-19

## `nuxy-two-panel` — ratio-based split API

- Replaced pixel `min-left` / `min-right` and `split` attributes with ratio-based `min-scale` (default `1/4`) and `default-position` (default `1/2`).
- `min-scale` sets the minimum width fraction for each panel; e.g. `1/4` allows dragging between 1/4 and 3/4, `1/10` between 1/10 and 9/10. At `1/2` the split is fixed at center and the handle is hidden.
- `default-position` controls the initial left-panel width on first mount (`1/4`, `0.25`, `25%` supported).
- `split-change` event now emits `{ ratio: number }` instead of `{ width: number }`.
- Added `parseRatio` utility in `extensions/ui-default/src/utils/parseRatio.ts`.
- Updated `settings` and `nyaa` extensions to use the new attributes.
