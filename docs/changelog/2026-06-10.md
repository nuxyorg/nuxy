# 2026-06-10

## Added

- **`core.shell.setSearchPlaceholder()`** — Active tools can override the omnibar placeholder at runtime via the shell bridge. Cleared automatically on `resetToolState()`.
- **Settings tool** — Omnibar shows “Search in settings”; typing filters the settings list by row label/description.
- **Nyaa tool** — Omnibar shows “Type anything to search in Nyaa” while the tool is active.

## Fixed

- **Preload script path** — Electron loaded stale `preload.mjs` while Vite builds `preload.js`, so `core.shell.setSearchPlaceholder()` was never exposed at runtime. Window manager now resolves the current preload bundle correctly.
