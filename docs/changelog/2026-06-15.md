# Changelog — 2026-06-15

## Added

- **`extensions/file-transfer/`** — new P2P file transfer tool extension (`com.nuxy.file-transfer`).
  - Sends files via WebRTC DataChannel with a shareable `FT-XXXX-XXXX` transfer code.
  - Uses PeerServer Cloud (`0.peerjs.com`) for signaling only — no PeerJS npm dependency; raw WebSocket + `RTCPeerConnection`.
  - Receiver enters code via omnibar; sender picks file with `nuxy-file-input`.
  - Live progress UI: percentage, speed, ETA, bytes transferred.
- Send screen uses list-based file picker (Enter/O) with proper tool layout sizing.

## Changed

- **`behavior.suppressBlurHide`** manifest flag — active tools can opt out of global `blurAction` (focus-out hide) while they are open.
- **Kernel** — `window.core.window.setBlurSuppressed()` IPC; shell applies it from manifest on tool activate/deactivate.
- **`com.nuxy.file-transfer`** — enables `suppressBlurHide` so native file picker focus loss does not close Nuxy.
  - Backend streams received chunks to disk via `core.fs` (`initReceive`, `writeChunk`, `finishReceive` IPC).
  - Settings: download directory, max file size, signaling host/port, STUN server.
  - Locales: English and Turkish.
  - Tests: `tests/utils.test.ts`, `tests/backend.test.ts`, `tests/frontend.test.ts`.
