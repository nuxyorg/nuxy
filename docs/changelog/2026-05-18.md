# Changelog - 2026-05-18

## Added

- Created `docs/changelog` directory for daily logging.
- Added daily changelog rules to `agents.md`.
- Root `.npmrc` with workspace link settings (`link-workspace-packages`, `shared-workspace-lockfile`).
- Root `packageManager` field and `preinstall` (`only-allow pnpm`) so installs always run via pnpm across all workspace packages.

## Changed

- Marked root `package.json` as `"private": true` (monorepo convention).
- Ignored `bun.lock` in `.gitignore` (Bun install at root does not install workspace packages; use `pnpm install` from repo root).
- Fixed `pnpm dev` crash: force CJS build for Electron main process (`vite.config.ts`) because `"type":"module"` made Vite emit ESM that cannot use named imports from `electron`.
