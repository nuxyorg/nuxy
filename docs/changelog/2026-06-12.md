# 2026-06-12

## Fixed

- **`nuxy-ext://` JSON imports** — Extension frontends that `import` `.json` files (e.g. Nyaa's `manifest.json`) no longer fail with a strict MIME type error. The protocol handler now serves JSON as `export default …` JavaScript modules.
- **Settings tool syntax** — Fixed a broken `.map()` callback in `settings/frontend.ts` that prevented the module from loading.

## Changed

- **Tool host layout** — Centralized flex column layout for tool content in `nuxy-tool-host > *`; removed duplicate `:host` flex styles from settings, nyaa, and notes extensions.
